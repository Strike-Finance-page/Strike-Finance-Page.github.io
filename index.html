What is Strike Finance?

Strike Finance is an on-chain perpetuals trading platform with a dedicated execution layer and cross-chain settlement, according to its official docs. The app is built for wallet-based trading, deposits, withdrawals, margin modes, order types, and API access.

How Strike Finance works

  1. Open the appUse the official app domain, then choose the live market or tool you need.
  2. Connect a walletConnect a supported crypto wallet and review any wallet prompts before signing.
  3. Deposit collateralUse the in-app deposit flow; validators quote, verify, and settle supported deposits on-chain.
  4. Manage positionsPlace orders, monitor margin, and use reduce-only, stop-loss, or take-profit controls where available.

Before you trade

Purpose-built execution

Trading activity runs through the Strike execution layer while settlement and custody are anchored to blockchain contracts.

Cross-chain settlement

The deposit and withdrawal docs describe validator-verified transfers across supported chains.

Trader and builder access

The official app serves active traders, while the documented API supports bots, dashboards, and custom integrations.

What is strike finance?

What can I do on strike finance?

You can access perpetuals markets, manage positions, use margin controls, deposit or withdraw supported assets, and review API resources for programmatic access.

Which networks does strike finance use?

The docs describe Cardano and Ethereum support for deposits and withdrawals, with settlement handled through on-chain locker contracts and validator verification.

Is strike finance risky?

Yes. Perpetuals, leverage, smart contracts, liquidations, and digital asset volatility can all create loss risk; read the official materials before trading.
